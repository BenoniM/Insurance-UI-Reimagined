# Workspace Guidelines for Git Pushes

- **Dual-Remote Push Requirement**: Always push changes to both remotes:
  1. `Insurance-UI-Reimagined` (https://github.com/BenoniM/Insurance-UI-Reimagined.git)
  2. `wass-insurance` (https://github.com/drogatechnology/wass-insurance.git)
- **Use origin**: The local git repository has been configured with an `origin` remote containing both push URLs. Run `git push origin <branch>` to push to both simultaneously.
